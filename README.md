# VoidBridge

**A zero-dependency, `nostdlib` interoperability layer between C/C++ and JavaScript.**

![Status](https://img.shields.io/badge/Status-Experimental-orange)
![License](https://img.shields.io/badge/License-MIT-blue)

## Disclaimer

> **This is a research project.** VoidBridge is built for educational purposes to explore the raw boundaries of WebAssembly.

## What is it?

Most WASM projects use Emscripten, which bundles a heavy standard library and runtime. **VoidBridge** is for the minimalists.
